## 🛠️ The Tech Stack

This showcase application is built using a modern, fast, and light-weight front-end development stack:

* **Framework**: **React 19** – Leveraging the newest features of functional components, robust hooks, and streamlined state lifecycle.
* **Language**: **TypeScript** – Utilizing strict interfaces, types, and safe import paths to ensure clean, bug-free, and highly maintainable application architecture.
* **Build System**: **Vite 6** – Offering ultra-fast compilation, fast HMR in code development, and optimized bundle sizes for instant load speed.
* **Styling & Theme**: **Tailwind CSS** – Implemented with full responsive wrappers (`sm:`, `md:`, `lg:`) and utility classes to build a modern cosmic-dark aesthetic:
  - Custom radial gradients for premium ambient background glow.
  - Consistent component curves (`rounded-xl`), border lines, and micro-hover scaling.
  - Elegant typography using the **Inter** sans-serif font family.
* **Icons**: **Lucide React & SVGs** – Crisp, high-contrast, perfectly-scaled vector icons and brand icons representing technical contact modes, code directories, and educational milestones.
